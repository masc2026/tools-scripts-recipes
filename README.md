# Sammlung für Werkzeuge, Skripte, Rezepte

Eigene Skripte und AI generierte Skripte (ChatGPT, Gemini). Für Archivierung und als Gedächtnisstütze.

-----

## Inhalt von diesem Repo

| Ordner | Zweck |
| :--- | :--- |
| [**VPN Access/**](vpnaccess/README.md) | Automatischer VPN Login (FU Hagen). Fragt nur nach Access Token. |
| [**Arch Linux Update/**](archlinupdate/README.md) | Sichere Prüfung für Arch Linux Updates. Nutzt temporäre Datenbank. |
| [**Migrate Git Repo/**](gitmigrate/README.md) | Lokales Git Repo zu neuem GitHub Repo verschieben. Schreibt Verlauf neu, vereinheitlicht Autoren und Daten. |
| [**Merge Policies/**](mergepolicy/README.md) | Vergleich von Git Strategien (Merge, Rebase, Squash). Zeigt Effekt auf Commit-Verlauf beim Zusammenführen. |
| [**New Repo Workflow/**](newrepoworkflow/README.md) | Neues Repo erstellen mit `gh` CLI. |
| [**Thunderbird PGP/**](thunderbirdpgp/README.md) | Gedanken zu Thunderbird und Digitale Signatur. |
| [**Arch Linux Migration to new SSD/**](archlinnewssd/README.md) | Bestehendes Arch Linux auf neue SSD umziehen. |
| [**Archive RAW/**](archiveraw/README.md) | RAW-Fotos (z.B. von SD-Karte) automatisch in Datumsordner kopieren. |
